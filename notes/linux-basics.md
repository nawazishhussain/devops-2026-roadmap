# Linux Basics – Week 1

## Commands Practiced

pwd – print working directory  
ls – list files  
cd – change directory  
mkdir – create directory  
rm – remove files  
cp – copy files  
mv – move/rename files 

## Practical Exercises Performed

1. Created a directory:
   mkdir devops_practice

2. Navigated into directory:
   cd devops_practice

3. Created subdirectories:
   mkdir logs scripts backups

4. Created a test file:
   touch logs/app.log

5. Added content to file:
   echo "Hello DevOps" > logs/app.log

6. Viewed file contents:
   cat logs/app.log

7. Listed directory structure:
   tree
